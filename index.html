Aprendendo sobre seletores personalizados

Seletores personalizados

ID (#)

Único

CLASS (.)

Múltiplo

ID (#)

São identificadores para personalizar um elemento. No HTML é identificado com id = "..." e no CSS com hashtag #

CLASS (.)

São identificadores para personalizar mais de um elemento. No HTML é identificado com class = "..." e no CSS com um ponto .

Se quiser aplicar a Class em um trecho do texto utilize a Tag SPAN no HTML.

O id é único por página de HTML e a Class é multipla, sendo assim pode ser utilzada diversas vezes. Porém pode-se usar duas class no mesmo elemento, um id e uma class etc. Porém o id sobrepõe as configurações da class.

Algumas Pseudo classes (:)

  • Active: Ativo (link que foi clicado).
  • Checked: Marcado.
  • Empty: Vazio.
  • Enabled: Habilitado.
  • Hover: Tenha uma reação quando passar o mouse em cima.
  • Focus: Foco.
  • Visited: Link que foi visitado.

    • Alguns Pseudo elementos (::)

  • After: Mostre algo depois.
  • Before: Mostre algo antes.
  • Backdrop: Pano de fundo (Um box que é renderizado imediatamente abaixo de qualquer elemento sendo apresentado no modo de tela cheia)
  • Content: Substitui um elemento por um valor gerado, os objetos inseridos são elementos substituídos anônimos.
  • Cue: Usado para estilizar legendas e outras sugestões em mídia com faixas VTT.

    • Child (>)

    Seleciona elementos com base em suas posições, o filho de um elemento.
